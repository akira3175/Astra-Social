# CHƯƠNG 3. CÀI ĐẶT – TRIỂN KHAI – KIỂM THỬ

## 3.1 MÔI TRƯỜNG PHÁT TRIỂN & CÔNG NGHỆ ÁP DỤNG
- **Hệ điều hành:** Chạy chéo nền tảng (Windows/macOS/Linux).
- **Môi trường Server Frontend:** Sử dụng Vite Build System (`npm run dev`) cổng mặc định 5173. Node.js >= 18.x.
- **Môi trường Server Backend:** PHP >= 8.2 bằng lệnh `php artisan serve` trên cổng 8000. Web Server (Apache/Nginx tùy chọn).
- **Cơ sở dữ liệu:** MySQL Server (XAMPP/MAMP/Docker tùy điều kiện).
- **Công cụ dòng lệnh:** NPM, Composer, PHP CLI, Git.

## 3.2 HƯỚNG DẪN CÀI ĐẶT & CHẠY DỰ ÁN (LOCAL ENVIRONMENT)

### 1. Chuẩn bị cơ sở dữ liệu (MySQL)
- Sử dụng PHPMyAdmin hoặc MySQL Client tạo một bảng trắng với tên `astra_social`.
- Đảm bảo cơ sở dữ liệu chạy ổn định trên port 3306.

### 2. Thiết lập Backend (Laravel)
- Di chuyển terminal vào thư mục `/backend`.
- **Cài đặt thư viện:** Chạy `composer install`.
- **Thiết lập Môi trường:** Tạo file `.env` bằng cách copy `.env.example`.
- **Sinh khóa bảo mật:** Chạy `php artisan key:generate`.
- **Khai báo Database:** Thiết lập `DB_DATABASE=astra_social` cùng user/password trong file `.env`.
- **Khởi tạo dữ liệu:**
  - Chạy `php artisan migrate` để tạo các bảng SQL tự động.
  - Chạy `php artisan db:seed` nếu cần khởi tạo tài khoản Admin và User gốc.
- **Khởi chạy ứng dụng Server:** `php artisan serve` -> (Listening on http://localhost:8000).

### 3. Thiết lập Frontend (React)
- Di chuyển terminal vào thư mục `/frontend`.
- **Cài đặt thư viện:** Chạy `npm install`.
- **Lưu ý Cấu hình APIURL:** Trong file cấu hình (`axios config`, file `.env`), kiểm tra biến `VITE_API_URL` trỏ về `http://localhost:8000/api`.
- **Khởi chạy Giao Diện:** Chạy `npm run dev` -> (Listening on http://localhost:5173/ hoặc URL tương ứng do Vite quyết định).

**Ghi chú:** Người chấm điểm có thể vào giao diện đăng ký để tạo mới trải nghiệm, hoặc dùng các tài khoản mẫu (nếu đã seed db).
